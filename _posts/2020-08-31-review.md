굿바이코로나 프로젝트는 2020.02.23에 시작된 코로나바이러스감염증-19에 대한 정보를 알려주는 서비스 입니다. 그 당시 코로나19에 대한 정보가 부족해 여러 사이트에서 정보를 습득해야했던 단점을 개선하고자 "코로나19" + "굿바이"가 결합된, 코로나19가 빨리 완치되었으면 하는 마음으로 굿바이코로나 프로젝트를 시작했었습니다.

![](https://raw.githubusercontent.com/dhlife09/dhlife09.github.io/master/_posts/images/2020-08-31-covid/1.png?token=AFIBA5G4QJNDVBZD7SWK7C27QLJQQ)


🚥 **현재 굿바이코로나 프로젝트에서 운영중인 서비스**

-   [Goodbye-Corona](https://corona-19.kr) - 코로나19에 대한 다양한 정보 직관적으로 보여드립니다.
-   [Corona-19-API](https://api.corona-19.kr) - 개발자를 위한 코로나19 국내 API 서비스를 제공합니다.


😞 **굿바이코로나 프로젝트를 운영하면서 힘들었던 점**

초기에는 사이트를 운영하는데 오류도 많이 있었고 본 사이트를 카피해서 운영하는곳도 있었으며 이걸 운영해도 될까? 하는 등 여러 고민이 있었지만, 주변에서 많은 분들께서 응원해주셨고, 본 사이트가 도움이 되었다는 메시지 하나하나가 프로젝트를 지속할 수 있는 큰 힘이 되어 많은 도움이 되었습니다.

지금까지도 많은 응원 덕분에 굿바이코로나 프로젝트를 이어나가고 있습니다

🔥 **더 많은 서비스에서 코로나19 정보를 사용할 수 있도록. Corona-19-API 서비스 시작**

![](https://raw.githubusercontent.com/dhlife09/dhlife09.github.io/master/_posts/images/2020-08-31-covid/2.png?token=AFIBA5DVRM5P6ZSIOX3FGVK7QLJTO)
Corona-19-API 서비스는 20/04/06에 시작된 더 많은 서비스(프로그램)에서 코로나19에 관한 정보를 사용할 수 있게 해주는 국내 코로나19 API 서비스 입니다. 직접 굿바이코로나 사이트를 운영하면서 보다 쉽게 많은 분들께 도움이 될 수 있도록 서비스를 시작하였습니다.

요청 카운터 제한없이 무제한 무료로 API 서비스를 제공중이며 월 평균 237,000건의 데이터를 처리하고 있습니다.

🧭 **본 Corona-19-API 서비스로..**
-   [**XpressEngine용 Widget이**](https://xetown.com/topics/1381911) 개발되었습니다.
-   [**마인크래프트 BE용 Plugin이**](https://github.com/bluefirewolf534/RCovid) 개발되었습니다.
-   [**광주광역시 감염병관리지원단 홈페이지에서**](http://gcidc.or.kr) 본 API 서비스를 사용하였습니다. (현재는 사용하지 않습니다.)
-   **학교 수업 목적으로도** 사용되었습니다.


![](https://raw.githubusercontent.com/dhlife09/dhlife09.github.io/master/_posts/images/2020-08-31-covid/3.png?token=AFIBA5DZMYEJIP5FKKVDBMK7QLJUM)

추가로 2020.05.01부터 2020.05.31까지 짧은 기간이었지만, Corona-19-API 서비스가 네이버 클라우드 플랫폼의 서버 지원을 받았었습니다. ~~(서버 반납기간이 하루 늦어서 2천원을 사비로 지불했었습니다... 좋은 만큼 비싸요.. ㅋㅋ )~~

📚 **굿바이코로나 프로젝트를 이어가며 배운점/느낀점**

-   혼자서 하는것도 좋지만 팀을 꾸려 같이 하면 더 좋을 수 있다는 것을 느꼈습니다.
-   이러한 사이트도 해킹 공격이 많이 온다는 것이 신기했습니다.
-   보안의 중요성과 여러 공격방법/방어방법을 알게되었습니다.
-   프로그래밍 언어에 조금 더 가까워졌습니다.
-   서버 운영/네트워크에 더 가까워졌습니다.

😷 **마치며** 

코로나19가 완치되어 하루빨리 건강한 세계로 돌아갔으면 좋겠습니다. (굿바이코로나는 코로나가 완치될까지 가능한 계속 운영할 예정입니다.)

코로나19 확산 방지를 위해 전국에서 노력해주시는 모든분들께 진심으로 감사드립니다! 👍
